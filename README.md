# Jua X - Travel & Booking App

A cross-platform Flutter application for travel and booking services within the city.

## Features

- Modern, beautiful UI with gradient designs
- Search functionality for destinations and services
- Category-based navigation
- Featured services showcase
- Popular destinations listing
- Bottom navigation bar

## Getting Started

### Prerequisites

- Flutter SDK (3.0.0 or higher)
- Dart SDK

### Installation

1. Install dependencies:
```bash
flutter pub get
```

2. Run the app:
```bash
flutter run
```
